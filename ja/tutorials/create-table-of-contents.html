--- layout: page title: チュートリアル⑦目次の作成 lang: ja --- {% capture content %} ## このチュートリアルの目標 - 目次を自動で作成できる - 目次を手動で作成できる - `target-counter()` を使ってページ番号付き目次を作成できる ## 目次の自動作成 Vivliostyle の機能を使って目次を自動で作成することができます。このとき目次に表示されるのは、 - 本のタイトル - 目次ページのタイトル（デフォルトは "Table of Contents"） - 各原稿ファイルのタイトルとリンク です。vivliostyle.config.js に以下の項目を追記します。 ```js // @ts-check import { defineConfig } from '@vivliostyle/cli'; export default defineConfig({ // ... toc: { title: '目次', }, }); ``` 以下のコマンドを実行します。 ```bash npm run preview ``` すると、PDF の1ページ目に目次が自動で追加されます。`toc.title` の値を変更することで、目次ページのタイトルを自由に変更できます。 ![](/assets/tutorials/ja/create-table-of-contents/preview-table-of-contents1.png) ### 原稿ファイルの各見出しを目次に表示したい場合 目次には原稿ファイルへの参照以外にも、原稿ファイル内の見出しも含めることができます。そのようにしたい場合、`sectionDepth` に `1` から `6` の値（それぞれ `h1` から `h6` までのどのレベルを含めるか）を指定します。次の例では、`h1` から `h3` の見出しを目次に含めるようにしています。 ```js toc: { title: '目次', sectionDepth: 3, } ``` 参照：[Vivliostyle CLI ドキュメント](https://docs.vivliostyle.org/ja/cli/toc-page/) ## 目次の手動作成とカウンタの参照 手動で目次ファイルを作成する場合、以下の手順で行います。 テキストエディタを開いて manuscripts/toc.md という名前の目次ファイルを作ってみましょう。role 属性の値 "doc-toc" は、[Digital Publishing WAI-ARIA](https://www.w3.org/TR/dpub-aria-1.0/) という仕様で定義されてます。Vivliostyle は、この属性のある要素（通常は nav）を目次として認識して、リンクされている HTML をまとめて一冊の本のように扱い、プレビュー画面の目次パネルのナビゲーションも有効にします。 ```html
1. [まえがき](maegaki.html) 1. [本文](honbun.html) 1. [ソースコード](honbun.html#code) 1. [キャプション付き](honbun.html#code-with-caption) 1. [脚注](honbun.html#footnote) 1. [Frontmatter](honbun.html#frontmatter) 1. [改行](honbun.html#newline) 1. [画像](honbun.html#figure) 1. [キャプション](honbun.html#figure-with-caption) 1. [数式](honbun.html#math) 1. [HTML](honbun.html#html) 1. [Markdown と HTML の併用](honbun.html#html-with-markdown) 1. [ルビ](honbun.html#ruby) 1. [あとがき](atogaki.html)
``` 目次ファイルを独自に作る場合は、vivliostyle.config.js の entry に追記が必要です。目次ファイルには `rel: 'contents'` を指定してください。 ```js // @ts-check import { defineConfig } from '@vivliostyle/cli'; export default defineConfig({ // ... entry: [ { path: 'toc.md', rel: 'contents', }, 'maegaki.md', // ... ], }); ``` 目次用の CSS も追記します。目次の各項目にリンク先のページ番号を自動挿入するには、`target-counter()` 関数を使います。`target-counter(attr(href), page)` は、各リンクの参照先要素が配置されているページのページ番号を返します。 ```css #toc li { list-style: none; } #toc li ol { padding-left: 1em; } #toc li a { display: inline-flex; width: 100%; text-decoration: none; color: currentColor; align-items: baseline; } #toc li a::before { margin-left: 0.5em; margin-right: 0.5em; border-bottom: 1px dotted; content: ''; order: 1; flex: auto; } #toc li a::after { text-align: right; content: target-counter(attr(href), page); align-self: flex-end; flex: none; order: 2; } ``` ![](/assets/tutorials/ja/create-table-of-contents/preview-table-of-contents2.png) ## まとめ このチュートリアルでは、目次を作成してそのスタイルを定義しました。今回のチュートリアルで使用した vivliostyle.config.js、原稿ファイル、スタイルファイルは以下のボタンからダウンロードできます。
